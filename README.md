# 🎬 Movie Recommendation App

[[Nomad Coders] ReactJS로 영화 웹 서비스 만들기 ](https://nomadcoders.co/react-for-beginners)에서 만든 프로젝트를 수정 및 확장한 프로젝트

<br/>

## 주요 기능

### ⭐ 최신순 또는 레이팅순으로 영화 20개를 추천

### ⭐ 추천된 영화 목록에서 제목으로 영화 찾기

### ⭐ 다크모드 가능

### ⭐ 반응형 웹
